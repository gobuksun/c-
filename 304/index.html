문장(Statement)의 종류#

  • 표식문 (Labeld Statement)

    • 문장 앞에 표식(label)이 붙는다.
    • goto문의 이동위치 지정에 사용
    • switch문과 함께 쓰이는 cas, default label이 있다.

  • 수식문 (Expression Statement)

    • ; 으로 끝나는 대부분의 문장 (치환문, 함수호출 ...)
    • 수식이 없는 문장을 공문(null statement)이라 한다.

  • 복합문 (Compund Statement)

    • 한개 이상의 문들의 나열이 { 와 }로 둘러싸인 형태로 block이라고도 한다.
    • 함수 정의 구조도 일종의 복합문이다.

  • 선택문 (Selection Statement)

    • if, if else, switch

  • 반복문 (Iteration Statement)

    • while, do while, for

  • 분기문 (Jump Statement)

    • goto, continue, break, return

  • 선언문 (Declaration Statement)

    • 프로그램에서 사용하고자 하는 대상의 명칭을 지정한다.
    • C++에서는 실행문의 위치에서도 선언이 가능하다.

  • for statement#
    1. [형식]
    2. for (초기식; 조건식; 증감식)
    3.    statement
    • 초기식 : 루프제어 변수를 설정하기 위한 지정문
    • 조건식 : 어떤 값과 루프제어 변수를 비교하여 언제 루프가 끝날지를 결정하는 관계연산자가 사용
    • 증감식 : 각 루프가 반복될 때마다 루프제어 변수를 변화시키는 방법을 정의
    • for 루프는 조건이 참인 동안 계속 반복된다.
    • 조건 비교에 실패하면 for 다음의 실행문으로 제어를 넘긴다.
    1. for (count = 1; count < num; count++)
    2.     cout << "continue";
    •  

    • for statment의 융통성

      • 루프 제어 변수의 데이터형에 제한이 없다.

        • 정수형 변수, 실수형 변수, 문자형 변수, 포인터 변수

      • 증감식에 제한이 없다.

        • 산술 급수적 증감, 기하 급수적 증감, 복합적 증감

      • 조건식에 제한이 없다.

        • 관계식, 논리식, 논리적 의미를 가지는 임의의 수식
      • 초기식에 제한이 없다.
      • 초기식, 조건식, 증감식은 생략 가능하다.
      • 루프의 반복횟수를 실행중에 변경 가능하다.
      • 여러개의 제어변수를 동시에 사용 가능하다.
      • break문 및 contine문의 제어를 받는다.

분기문#

  • break statement#
    • for, while, do-while, switch 문에 대하여 영향력을 갖는다.
    • break 문을 기준으로 하여 가장 가까운 블록 또는 반복구조에서 한 겹만 빠져 나온다.

 

  1. #include <iostream>
    using namespace std;

    void func1(void)
    {
        int n, sum = 0;
       
        for (n = 1; ; n++) {
            sum += n;
            if (500 < sum)
                break;
        }
        cout << "Sum => " << sum << endl;
    }

    void func2(void)
    {
        int i, j;
       
        for (i = 2; i < 1000 ; i++) {
            for (j = 2; j <= i/2; j++) {
                if (!(i%j))
                    break;
            }
            if (j > i/2)
                cout << i << " is prime" << endl;
        }
    }

    void main(void)
    {
        func1();
        func2();
    }

 

  • continue statement#
    • for, while, do-while 문에 대하여 영향력을 가진다.
    • continue 를 만나면 해당 Block의 나머지 문장은 Skip하고 반복구조의 처음으로 돌아간다.
    • for 문에서는 증감식으로 제어 이동한다.

 

  1. #include <iostream>
    using namespace std;

    void func1(void)
    {
        int s = 0;
        for (int i = 1; i <= 100; i++) {
            if (i % 5 == 0)
                continue;
            s += i;
        }
        cout << s << endl;
    }

    const int Size = 80;
    void func2(void)
    {
        char line[Size];
        int  spaces = 0;

        cout << "Enter a line of test : \n";
        cin.get(line, Size);
        for (int i = 0; line[i] != '\0'; i++) {
            cout << line[i];        // 문자를 디스플레이
            if (line[i] == '.')        // 마침표이면 중지
                break;
            if (line[i] != ' ')        // 나머지는 루프를 건너뜀
                continue;
            spaces++;
        }
        cout << '\n' << spaces << " spacess\n";
    }

    void main(void)
    {
        func1();
        func2();
    }

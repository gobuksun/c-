C++ 프로그램 구조#

  • 선언, 정의, Preprocessor 로 구성

    • 선언 : 객체(변수), 함수에 대하여 명칭과 속성을 설정

    • 정의

      • 객체(변수) : 명칭, 형, 기억영역을 할당
      • 함수 : 명칭, 형, 형식매개 변수의 선언과 함수체부인 Block을 포함

    • 전처리기

      • # 으로 시작되는 행에 대하여 컴파일러가 가동되기 전에 행하는 처리과정

        • 파일 포함 : #include
        • 매크로 정의 : #define
        • 조건 컴파일 : #if, #else, #endif, #ifdef, #ifndef, #elif

         

  • 하나의 프로그램은 적어도 하나의 함수 정의(main 함수)를 포함
  1. #include <iostream>                    // preprocessor
    #define PI 3.14                        // preprocessor

    using namespace std;
    void main(void)                        // main 함수 정의
    {
        float radius;                    // 변수 정의
        float circle_area(float);        // 함수 선언
       
        cin >> radius;                    // 객체에 message 전달
        cout << circle_area(radius);    // 객체에 message 전달
    }

    float circle_area(float r)            // 함수 정의
    {
        return (r * r * PI);
    } // end of circle_area

C와 C++의 표현 비교#

  1. C 의 표현 C++ 의 표현
    #include <stdio.h>

    int main(void)
    {
        float kor, mat, eng, aver;
        char name [20];

        printf("Enter a name : ");
        scanf("%s", name);
        printf("Enter Scores : ");
        scanf("%f%f%f", &kor, &mat, &eng);
        aver = (kor + mat + eng) / 3.0;
        printf ("Name : %s\nAverage : %f", name, aver);
    }    		 #include <iostream>
    using namespace std;

    int main(void)
    {
        char name [20];
        cout << "Enter a name : ";
        cin >> name;

        float kor, mat, eng, aver;
        cout << "Enter Scores : ";
        cin >> kor >> mat >> eng;

        aver = (kor + mat + eng) / 3.0;
        cout << "Name : "  << name << "\nAverage : " <<  aver;
    }

 

C++ 의 형식#

  • 특정한 폼이 없는 Free Form Syntax
  • 되도록 한 줄에 하나의 명령문을 사용
  • 각 명령문 뒤에는  ;  을 붙여 분리
  • 대문자 / 소문자 구분 (통상적으로 Keyword는 소문자, 매크로는 대문자)
  • 되도록 들여쓰기(indentation)하여 프로그램을 읽기 편하게 기술

 

main 함수#

  • 프로그램은 하나 이상의 함수로 구성
  • 함수 중에는 반드시 main 함수가 존재
  • 프로그램의 실행 주체가 되는 함수
